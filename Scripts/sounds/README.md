# Sound Files for Pattern Memory Game

This directory should contain the following sound files for the game to have audio:

## Required Sound Files:

1. **red.mp3** - Sound for red button
2. **blue.mp3** - Sound for blue button  
3. **yellow.mp3** - Sound for yellow button
4. **green.mp3** - Sound for green button
5. **success.mp3** - Sound played when completing a sequence correctly
6. **error.mp3** - Sound played when making a mistake
7. **combo.mp3** - Sound played when achieving combo milestones

## Where to Get Sounds:

You can find free game sound effects at:
- [Freesound.org](https://freesound.org/)
- [Zapsplat.com](https://www.zapsplat.com/)
- [Mixkit.co](https://mixkit.co/free-sound-effects/game/)

## Recommended Sounds:

- **Button sounds**: Short beeps or tones (each button should have a different pitch)
  - Red: Low tone (e.g., C4 - 261 Hz)
  - Blue: Mid-low tone (e.g., E4 - 329 Hz)
  - Yellow: Mid-high tone (e.g., G4 - 392 Hz)
  - Green: High tone (e.g., C5 - 523 Hz)
- **Success**: Positive chime or "level up" sound
- **Error**: Buzzer or negative beep
- **Combo**: Energetic whoosh or power-up sound

## Note:

The game will continue to work without sound files - audio errors are caught gracefully. However, sound greatly enhances the gaming experience!
